### Getting Started

#### Prerequisites

-	[Node.js](https://nodejs.org/en/download/)

#### "Installation"

```
npx brianjenkins94/kerplow [--update|-u] [--yes|-y]
```

_or_

```
npm install brianjenkins94/kerplow --production
```
